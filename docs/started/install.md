# Installation


##  Get it on NuGet!
***

```bash
> Install-Package JinianNet.JNTemplate

```

or

```bash
> dotnet add package JinianNet.JNTemplate
```


## Building the Source
***
- WINDOWS:
```batch
> git clone https://gitee.com/jiniannet/jntemplate.git
> build/build.bat
```

- LINUX:
```bash
> git clone https://gitee.com/jiniannet/jntemplate.git
> cd build
> ./build.sh
```
